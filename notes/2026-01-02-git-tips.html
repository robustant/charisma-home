📅 2026-01-02 🏷️ git, 工具

1. 撤销最近一次提交（保留工作区修改）

git reset --soft HEAD~1

2. 强制推送（慎用！）

git push --force-with-lease

3. 查看某次提交的变更文件

git diff --name-only HEAD~1

4. 忽略已跟踪文件的修改

git update-index --assume-unchanged <file>

“Git 是时间机器，不是备份工具。” —— 习惯用 git log --oneline 回溯历史。
